## :rocket: Getting Started

1. Download the repository

  - Using Git
```shell
  git clone https://github.com/gledson-ss/teto-solution.git
```
  - Using Github CLI
```shell
  gh repo clone gledson-ss/teto-solution
```
  > :bulb: or whichever way you prefer

2. Installation
  - Using `yarn`
   ```shell

   ```
  - Using `npm`
   ```shell
   npm install
   ```

  - Using `yarn`

  ```shell
   yarn start
  ```
